논문 및 부가 자료

학과 제출 규정에 맞추어 논문 파일, 개발한 시스템 코드, 시연 동영상, Testing data를 정리할 수 있도록 별도 페이지를 구성하였다.

자료 목록

  • 논문 파일 최종 논문 PDF를 연결하는 항목.
  • 개발한 시스템 코드 압축 파일 또는 저장소 링크로 교체 가능.
  • 시연 동영상 동영상 파일 또는 외부 링크로 교체 가능.
  • Testing data 테스트용 입력 자료 및 검증 데이터 업로드용 항목.

업데이트 방법

  • docs/downloads/ 폴더 안의 placeholder 파일을 실제 자료로 교체하면 된다.
  • 파일명을 그대로 유지하면 HTML 링크 수정 없이 바로 적용할 수 있다.
  • 외부 저장소나 동영상 플랫폼 링크를 사용할 경우 해당 링크 주소만 이 페이지에서 바꾸면 된다.
